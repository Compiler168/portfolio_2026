# Personal Portfolio Website - Frontend

Modern, responsive frontend for Majid Iqbal's portfolio website.

## Tech Stack
- HTML5 / CSS3 (Modern, responsive design)
- Vanilla JavaScript (ES6+)
- Font Awesome (Icons)
- Google Fonts (Outfit, Inter)

## Key Features
- **Premium Design System**: Sophisticated dark-mode glassmorphic UI.
- **Scroll Animations**: Dynamic reveal effects using Intersection Observer.
- **Dynamic Content**: Data-driven portfolio, skills, and testimonials.
- **Full Responsiveness**: Optimized for mobile, tablet, and desktop.
- **Interactive Modals**: Detailed project views with case study info.

## Setup & Running
1. Navigate to this directory:
   ```bash
   cd frontend
   ```
2. Start a local server:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser.
- React 18
- Tailwind CSS
- Axios for API calls
- Framer Motion for animations

## Features
- Responsive design (mobile, tablet, desktop)
